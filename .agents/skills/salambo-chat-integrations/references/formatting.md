# Chat Formatting Rules

Use this format for chat-first agents:

1. Direct answer.
2. Brief supporting bullets.
3. Clear next action if needed.

## Prefer

```text
Yes — I can do that.

I need one thing: which workspace should I use?
```

## Avoid

- long essays;
- huge tables;
- hidden implementation details;
- raw JSON unless requested;
- dumping large generated files into chat.
