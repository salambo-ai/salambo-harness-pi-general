# Prompt Template Guidance

Prompt templates live under:

```text
agent/prompts/
```

Use them for reusable prompt text that should not be embedded directly in the system prompt.

Good uses:

- report outlines;
- review rubrics;
- extraction templates;
- domain-specific checklists.

Rules:

- Keep templates plain text or Markdown.
- Name files by purpose.
- Keep examples sanitized.
- Do not include secrets or customer-specific private data.
